## Run Locally

Clone the project

```bash
  git clone https://github.com/JuviGamez/open-calculator.git
```

Go to the project directory

```bash
  cd open-calculator
```
Make node_modules
```bash
  npm install
```

Install dependencies

```bash
  npm install react react-dom styled-components framer-motion
```

Start the server

```bash
  npm run dev
```

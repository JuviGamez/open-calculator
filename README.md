## Run Locally

Clone the project

```bash
  git clone https://github.com/JuviGamez/open-calculator
```

Go to the project directory

```bash
  cd open-calculator
```

Install dependencies

```bash
  npm install react react-dom styled-components framer-motion
```

Start the server

```bash
  npm run dev
```
